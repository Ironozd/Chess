\doxysubsection{Detailed Description}
Logical Device singleton. 

\begin{DoxyNote}{Note}
Since Vulkan Vk\+Device (\doxylink{class_renderer_1_1_logical_device}{Logical\+Device}) is expected to be created and destroyed once and Queues that are going to be accessed throughoutare created depending on this class, this class was made a singleton. 

This device can be created more than once, however, for the current use case, singleton appeared to be the better fit. 
\end{DoxyNote}
